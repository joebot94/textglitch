## [0.1.0] - 2026-03-11
### Added
- Dynamic grid profiles (1x1, 2x2, 3x3, 3x4, 4x4)
- Preset-aware rendering and custom tile selection improvements
- OSC controls for grid switching and blank/unblank tile control
- Auto preset choreography with sequential/random modes and anti-repeat random policy
- Live controls for auto switch mode/interval/preset pool
- Keyboard performance shortcuts for auto toggle and preset stepping

### Changed
- Refactored preset switching into a dedicated controller module for future beat-sync and scene workflows
